FractionalDiffEq.ChebSpectral ‚Äî Type
solve(prob::SingleTermFODEProblem, n, ChebSpectral())

Using Chebyshev spectral method to solve single term fractional ordinary differential equations.

Warning

Chebyshev spectral method only support for linear fractional differential equations, and the time span of the problem should be $[-1, 1]$.

References

@thesis{
    title = {Spectral Methods for Fractional Differential Equations}
    author = {Chinenye Assumpta Nnakenyi, J. A. C. Weideman, Nicholas Hale}
    year = {2015}
}

FractionalDiffEq.FiniteDiffEx ‚Äî Type

Usage

solve(Œ±, dx, dt, xStart, xEnd, n, Œ∫, FiniteDiffEx())
Tip

Here, if we set $0<\alpha\leq 1$, the equation is the fractional diffusion equation or subdiffusion equation, whereas for $1<\alpha\leq 2$, the equation is the fractional diffusion-wave equation.

Use explicit Caputo discretization method to solve the diffusion equations.
