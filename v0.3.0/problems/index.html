FODE Problem

Fractional ordinary problems type, we can classified them as Single-Term and Multi-Term problems:

\[D^{\alpha}y(t)=f(t, y)\]

\[\sum_{s=0}^pc_sD^{\beta_s}y=f\]

FractionalDiffEq.SingleTermFODEProblem — Type
SingleTermFODEProblem(f, α, u0, tspan)

Define a single term fractional ordinary differential equation, there are only one fractional differential operator in this problem.
